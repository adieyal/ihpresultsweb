WP Sidebar for Twitter, shows your latest tweets as a sidebar widget in your WordPress blog.

== Description ==

Display you latest twitter posts. This is the best twitter API for wordpress your will find. Please find more information about this twitter sidebar widget at <a href="http://www.dallasprowebdesigners.com" title="wordpress web designers">website developers</a> website.

**Features**

    * Simple to use
    * Easy to install
    * Customizable
    * Support available
    * Uses Wordpress resources (no extra files needed)
    * Option to detects URLs, e-mail addresses and replies
	
A complete list of features and other info can be found at <a href="http://www.dallasprowebdesigners.com/" title="Website Design Company"> Web Design Company</a> website that created this extension.
	
	
**Usage**

Once you have installed the twitter sidebar widget you need to go to your widgets section of your blog and simply drag and drop the twitter sidebar widget into the sidebar you wish to have it displayed in. Click the arrow to set tweeter display options and configure your twitter settings and you should be go to go. Have fun tweetering!

See the <a href="http://www.dallasprowebdesigners.com">Web Design Company</a> that developed this for further information. 
Developer documentation is also available for those who need instructions on how to make modifications.

== Frequently Asked Questions == 

=How many tweets can I display at a time?=

*You can show as many as you want and you have an option to tell the widget how many
you would like to display.


=Does it show my tweets in real time as I am tweeting them?=

*Yes it shows them in real time. If you are tweeting now you will have to refresh your
website for the to appear but they will be there once you do.
